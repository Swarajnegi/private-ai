# AI/ML — Transferable Lessons

> A curated digest of every generalizable lesson distilled from JARVIS build work, ML side-projects, and production AI/ML systems the user has touched. Project-specific entity/model/stack names have been stripped or demoted to examples. Every lesson cites its source (KB entry, file path, or session date) so you can dive back for full context.
>
> **Scope**: Concepts, patterns, gotchas, and principles that apply on *any* AI/ML engineering project — not tied to one codebase, one model family, or one framework.
>
> **How to read**: Each lesson is a short principle + why it matters + one concrete example. Read top-to-bottom for breadth, or jump to a section when you hit that problem on a future project.
>
> **Companion file:** [AI_ML_Interview_Prep.md](AI_ML_Interview_Prep.md) — interview-shaped articulation drills (Q&A bank). This file is for hard-won lessons; that file is for rehearsing how to explain them.
